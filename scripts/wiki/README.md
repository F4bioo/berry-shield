# Wiki Factory Tools

This directory contains the automation logic for the Berry Shield technical documentation.

## Core Structure
- **[/scripts](./scripts)**: Automation engine (`doc-factory.ts` and `doc-sanity.ts`).
- **[/templates](./templates)**: Markdown templates for Diátaxis pages (Reference, Explanation, etc.).
- **[/references](./references)**: Technical specifications (e.g., Diátaxis and project layers).

## Usage
Refer to the main [scripts/README.md](../README.md) for execution instructions.
